# Logseq 插件：Left-Sidebar Enhance

> 一个 Logseq 插件，增强左侧边栏功能：页面大纲（目录）、标题层级编号、自动标题等级调整、鼠标悬停唤出侧边栏等。

## 功能

### 1. 页面大纲（目录）

根据当前页面的标题（`#` 到 `######`）自动在左侧边栏生成目录。

- **单击** → 跳转到标题位置
- **Ctrl+单击** → 以缩放页面打开
- **Shift+单击** → 在右侧边栏中打开
- **悬停高亮** → 鼠标悬停目录项时高亮对应数据块；聚焦某数据块时高亮其目录项
- **日记支持** → 在日记页面显示日期列表（可在设置中开关）

### 2. 标题层级编号（仅限文件型图谱）

通过工具栏按钮手动为 Markdown 标题添加层级编号（如 `1`、`1.1`、`1.1.1`）—— 纯按钮触发，无后台监听，无性能损耗。

- **工具栏按钮** → 点击「重新编号」为当前页面编号
- **块右键菜单**：
  - **跳过** — 跳过此标题，不参与编号；其子标题继承父级编号前缀
  - **锁定** — 保持当前编号不变；后续同级标题从该编号递增
  - **重号** — 与上一个同级标题使用相同编号
- **按图谱持久化** → 标题状态存储在图谱的 assets 目录中，重启后保留
- **安全清理** → 启动 15 秒后自动清理已删除块的残留数据

### 3. 自动调整标题等级（仅限文件型图谱）

根据块树的缩进深度自动规范 Markdown 标题等级，适用于 H1–H4 范围。

- **保留 H1** → 可选择将 H1 保留给页面标题
- **命令支持** → 通过命令面板（`Ctrl+Shift+P`）搜索「Normalize headings」即可规范化当前页面或选中区域的标题

### 4. 鼠标悬停唤出侧边栏

左侧边栏隐藏时，将鼠标移至左边缘即可临时唤出。

- **A 型** → 靠近左上角区域触发
- **B 型（推荐）** → 鼠标移到窗口最左列触发
- 通过左上角 `≡` 按钮或设置开关切换

### 5. 收藏夹和历史记录去重

启动时及每 10 分钟自动删除 Favorites 与 History 中的重复项。

## 安装

### 通过 Logseq 市场

1. 点击工具栏的 `•••` 按钮，选择「插件」
2. 切换到「市场」标签
3. 搜索 `Left-Sidebar Enhance` 并安装

### 手动安装

1. 从[发布页](https://github.com/YU000jp/logseq-plugin-left-sidebar-enhance/releases)下载最新发布包
2. 解压到 Logseq 插件目录
3. 在 Logseq 插件管理中启用

---

## 使用说明

- **页面大纲** — 打开任意页面，目录自动显示在左侧边栏
- **标题编号** — 点击页面工具栏的 `[1 2 3]` 图标重新编号；右键点击标题块圆点可设置跳过/锁定/重号
- **自动调整等级** — 通过命令面板（`Ctrl+Shift+P`）搜索「Normalize headings on current page」或「Normalize headings in selection」
- **鼠标悬停** — 点击左上角 `≡` 按钮在隐藏模式和悬停模式间切换

---

## 插件设置

| 设置项 | 说明 |
|--------|------|
| 隐藏收藏夹和历史记录中的重复项 | 启动时及每 10 分钟自动去重 |
| 鼠标悬停唤出侧边栏 | 启用/禁用，选择 A 型或 B 型 |
| 页面大纲（目录） | 启用/禁用，悬停高亮，日记日期列表 |
| 标题编号 | 按钮触发，通过右键菜单配置跳过/锁定/重号 |
| 自动调整标题等级 | 启用/禁用，保留 H1（标题范围固定 H1–H4） |

---

## 数据存储

标题状态（跳过/锁定/重号）以 JSON 文件形式存储在图谱的 `assets/storages/left-sidebar-enhance/` 目录下，使用 Logseq 的 SandboxStorage API。数据按图谱隔离，在 Logseq 页面列表中不可见。
